# The Shadow Cat

## Synopsis

The Shadow Cat is a top-down action roguelike where you play as a small, agile black cat on a quest to challenge rival feline factions and restore balance to the kingdom. Each run is made of handcrafted rooms filled with randomly placed enemies, obstacles and items; runs end when the player loses all lives. Progression happens via items and ability upgrades found during runs.

## How to play

- Movement: W / A / S / D (or left stick)
- Interaction: F (keyboard) / Y (Xbox)
- Basic attack: Left mouse button (or X on Xbox)
- Charged attack: Hold left mouse button (or hold X)
- Special attack: Right mouse button (or B on Xbox)
- Dodge: Shift (keyboard) / RT (Xbox)
- Pause: Esc (keyboard) / Start (Xbox)

## Game rules (quick)

- You start each run with a limited number of lives. Taking damage reduces lives; run ends when lives reach zero.
- Enemies drop items (active or passive). Active items can be used; passive items modify player stats.
- Rooms are handcrafted but populated randomly. Defeat the boss at the end of a phase to progress.
